| b | No *a* before direct-object people (*chiamo Maria*, not *chiamo a Maria*) | Direct Spanish *a personal* interference. |

| d | Futuro `-care/-gare` h-insertion and `-iare` merge | Still slightly leaky — resurfaced in a time-telling drill: *mangiare → mangieremo* instead of *mangeremo* (kept the stem *-i-* that should drop before *-e/-i* endings). |
| e | **Verb government: direct vs indirect object per verb** (which verb takes bare object vs *a*) | Self-reported hardest area — clitic *mechanics* are solid, the leak is purely *lexical*: knowing a given verb's category. Trap verbs: *telefonare/rispondere/chiedere* → indirect (Sp *llamar/preguntar* mislead); *ringraziare/pagare/aspettare/ascoltare/cercare/guardare* → direct (no *a*); *pensare a/tenere a/contare su* → neither clitic, needs the right preposition. Current slips: *ringraziare* still taken as indirect; *pensare* still reached for *di* instead of *a*; *chiedere* double-object drops the direct *lo* (says only *gli*, should be *glielo*); *contare su* produced as *contare in*. Same pattern now also surfaces in *ne* vs *ci* choice under *pensare a* (topic) vs *pensare di* (opinion). |
| f | **Irregular past participles regularized to `-uto`** | *chiedere → chieduto* (should be *chiesto*), *rispondere → risposta* (should be *risposto*) — *-uto* over-generalized as default. |
| g | *-cia/-gia* plural: consonant-preceded subclass (drop *i*) | *arancia→arancie* instead of *arance* — applied the vowel-preceded retention rule (*camicia→camicie*, correct) to a consonant-preceded case where it should drop. |
| h | Numerals: *milione* construction and thousands spelling | *Mille* used for "a million" with *di* dropped entirely (should be *un milione **di** persone*); *due mille* written as two words instead of *duemila*; *ventitresimo* missing the double-*e* (*tré + esimo → ventitre**e**simo*). Elision (*ventuno, ventotto*) and *cento* (invariable, no article) were solid. |
| i | Dates: dropping the article entirely when there's no explicit "on"-preposition | *Tre ottobre partiamo...* instead of *Il 3 ottobre partiamo...* — correctly saw no separate preposition is needed, overcorrected by dropping the article too. Rest of the date system (one-off vs habitual day + article, *domenica* fem., *il primo* exception, cardinal not ordinal) was solid. |

## Standing interference watchlist

Patterns to flag in any Italian Daniel writes.

**Syntax**
- *a* before direct-object people (a-personal carryover).

**Vocab slips**
- *bono* → *buono*
- *agora mismo* → *adesso* / *ora*
- *cada vez* → *ogni volta*
- *otra vez* → *di nuovo* / *un'altra volta*

**Morphology**
- Plural family members need the article: *i miei fratelli*, not *mio fratelli*.

## How to maintain this file

- Add a row once an error type **recurs** (2+ instances) in quiz or free writing, or Daniel flags it as a real struggle. A single isolated slip — even on brand-new material — is normal noise, not a pattern, and does **not** get a row.
- **Overwrite, don't append.** When a row's state changes, replace its note with the current picture — never stack a new dated paragraph next to the old one. Each row is a snapshot of where things stand *now*, not a log of every re-drill.
- Remove a row entirely once a clean drill confirms internalization — don't leave a "resolved" note behind.
- Keep it short and clinical. Drill ideas, sequencing, and planned additions belong in `LEARNING_ROADMAP.md`.
